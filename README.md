Viewer for large SVGs generated by Graphviz - call graphs, control flow graphs, and other visualizations common in reverse engineering, compiler engineering, and profiling.

## Features:
- Native browser search (ctrl+f), text selection
- Native browser scrolling and touch pan/zoom
- Regex text search
- Minimap
- Scroll wheel zoom
- Dark mode

## Embedding SVGs

You can use `index.html` as a template to create self-contained HTML files with an embedded SVG. This is useful for generating standalone viewers that can be easily shared or opened locally.

### Usage

1. Copy `index.html`
2. Replace `<!-- SVG_CONTENT -->` with your SVG markup

### Example

```bash
# Using sed
sed 's/<!-- SVG_CONTENT -->/'"$(cat my-graph.svg)"'/' index.html > my-graph.html
