# SVG Viewer

https://trumank.github.io/svg-viewer/

Viewer for large SVGs generated by Graphviz - call graphs, control flow graphs, and other visualizations common in reverse engineering, compiler engineering, and profiling.

## Features:
- Native browser search (ctrl+f), text selection
- Native browser scrolling and touch pan/zoom
- Regex text search
- Minimap
- Scroll wheel zoom
- Dark mode
